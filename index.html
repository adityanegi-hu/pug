Pug Template Engine

Pug is a high-performance template engine for Node.js that serves as an HTML preprocessor. It allows you to write cleaner, more concise code by replacing traditional HTML syntax with a whitespace-sensitive, indentation-based language that is eventually compiled into standard HTML.

Key Concepts of Pug

  • Cleaner Syntax: It eliminates the need for closing tags and uses indentation to define the structure of the document.
  • Dynamic Content: Since it is a template engine, you can inject variables, use conditionals (if/else), and perform loops directly within your files.
  • Reusable Components: It supports mixins, which allow you to create reusable blocks of code.
  • Node.js Integration: Commonly used with frameworks like ExpressJS to render dynamic web pages.

Why Use Pug?

Many developers choose Pug because it significantly reduces the amount of code you have to write compared to standard HTML. By focusing on indentation and removing verbose syntax, it leads to better readability and faster development.

History

Previously known as Jade, it was renamed to Pug in 2016 due to trademark conflicts. Tutorials referencing "Jade" refer to the same technology.

Example Comparison

Pug Code:

doctype html
html(lang="en")
  body
    h1 Welcome to Pug
    div.container
      p This is much shorter than HTML!

Compiled HTML Output:

<!DOCTYPE html>
<html lang="en">
  <body>
    <h1>Welcome to Pug</h1>
    <div class="container">
      <p>This is much shorter than HTML!</p>
    </div>
  </body>
</html>
